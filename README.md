# EnergyRL
Energy Usage in Relation to Reinforcement Learning
Python Verison: 3.10.11
numpy Version: 2.2.6

Inside the directory run:
`python3 -m venv venv` 
to create an virtual env.

run the virtual env by either using:
`source venv/bin/activate` on MacOS

or

`venv\Scripts\activate` on Windows

Install numpy using:
`pip install numpy`

then run:
`python TestEnergy.py`
